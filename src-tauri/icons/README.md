# Иконки приложения

В этой папке должны находиться следующие файлы иконок:

## Необходимые файлы:

- **32x32.png** - иконка 32x32 пикселей
- **128x128.png** - иконка 128x128 пикселей  
- **128x128@2x.png** - иконка 256x256 пикселей (для Retina дисплеев)
- **icon.icns** - иконка для macOS
- **icon.ico** - иконка для Windows

## Генерация иконок

Вы можете использовать онлайн-инструменты для генерации иконок из одного изображения:

1. **Tauri Icon Generator**: https://icon.kitchen/
2. Загрузите одно изображение (рекомендуется минимум 512x512 пикселей)
3. Скачайте сгенерированные иконки
4. Поместите их в эту папку

Или используйте инструменты командной строки:
- ImageMagick для конвертации
- iconutil для macOS (создание .icns)
- онлайн конвертеры для .ico файлов

## Временное решение

Если иконки отсутствуют, вы можете временно закомментировать секцию `icon` в `tauri.conf.json`:

```json
"icon": [
  // "icons/32x32.png",
  // ...
]
```

Но для финальной сборки иконки обязательны.
